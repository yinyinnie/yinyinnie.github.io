DataBinding之『简单使用』

  • Android

什么是Data Binding

DataBinding可以说是一种Android平台MVVM的框架，直接将数据和视图进行绑定。

Google很早就推出了Data Binding Library, 不过一直是Beta版，存在着一些Bug。 这次Google IO 大会（2016年）上发布最新Data Binding Library，现在DataBinding已经是Production了，你可以放心的玩耍了！！

使用Data Binding Library的环境要求

  • 它是Support Library，Android 2.1(ApI Level 7+) 都可以使用

  • Android Gradle Plugin 1.5.0-alpha1以上版本

  • Android Studio 1.3以上版本

简单使用步骤

Step0

新建一个Android Project，修改app module中的build.gradle文件，加入以下代码：

android {
    ....
    dataBinding {
        enabled = true
    }
}

注意： 如果你的工程依赖了一个使用了Data Binding 的Library，你的app module中build.gradle依然要加入以上代码

Step1

准备你的ViewModel类，有两种写法，如下样例：

第一种写法

/**
 * Created on by yydroid 8/17/16.
 */
public class User {
    private final String firstName;
    private final String lastName;
    public User(String firstName, String lastName) {
        this.firstName = firstName;
        this.lastName = lastName;
    }
    public String getFirstName() {
        return this.firstName;
    }
    public String getLastName() {
        return this.lastName;
    }
}

第二种写法

/**
 * Created on by yydroid 8/17/16.
 */
public class User {
    public final String firstName;
    public final String lastName;
    public User(String firstName, String lastName) {
        this.firstName = firstName;
        this.lastName = lastName;
    }
}

Step2

使用DataBinding时，布局文件有点不太一样，如下样例：

<layout xmlns:android="http://schemas.android.com/apk/res/android">
    <!--官方说法:这里不写,会自动生成MainActivityBinding类.
    但是我测试下,这里有个Bug,在我这版本的AndroidStudio(2.2 beta)中,这个类不会自动生成-->
    <data class="cn.onlinecache.databindingdemo.MainActivityBinding">
        <variable name="user" type="cn.onlinecache.databindingdemo.User"/>
    </data>
    <LinearLayout
        android:orientation="vertical"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:gravity="center">
        <TextView android:layout_width="wrap_content"
                  android:layout_height="wrap_content"
                  android:text="@{user.firstName}"
            android:textSize="40dp"/>
        <TextView android:layout_width="wrap_content"
                  android:layout_height="wrap_content"
                  android:text="@{user.lastName}"
            android:textSize="40dp"/>

    </LinearLayout>
</layout>

Step3

在你的对应Activity的onCreate，加入核心绑定代码，如下样例：

...
 @Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
     MainActivityBinding binding = DataBindingUtil.setContentView(this, R.layout.activity_main);
    User user = new User("YYDroid", "Nie");
    binding.setUser(user);

}
...

注意:

  • MainActivityBinding类是自动根据Step2中的配置来生成的
  • 当使用ListView或者RecyclerView时，在对应的Adapter类中加入代码：
ListItemBinding binding = ListItemBinding.inflate(layoutInflater, viewGroup, false);
//or
ListItemBinding binding = DataBindingUtil.inflate(layoutInflater, R.layout.list_item, viewGroup, false);
