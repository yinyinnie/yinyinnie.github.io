Mac说之『打造自己的命令终端』

  • Mac

Mac的GUI和命令行的完美结合让广大程序员们对它情有独钟。也正是因为这样，命令行的重要性可想而知。本文的重点就是告诉你如何打造自己的命令行。

第一步：下载iTerm

iTerm 这个工具我在Mac说之『我的工具集』中提到过，没安装的，赶紧下载安装吧，官网地址：https://www.iterm2.com/

第二步：切换成zsh

打开iTerm, 输入：chsh -s /bin/zsh

第三步：安装oh-my-zsh

  • 安装git (Mac系统自带，可以跳过)
  • git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh
  • cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc

第四步：配置zsh

完成以上三步，你的终端基本上配置完成了。

如果还需要按照自己的喜好进行配置，你只需要找到当前系统用户所在的目录，找到.zshrc文件，按照一定的要求配置这个文件，你就能打造自己的个性终端了。

浏览这个网页https://github.com/robbyrussell/oh-my-zsh/wiki/themes, 你能看到所有不同主题的预览。
