对于使用Mac工作的你，肯定在寻找一些比较好用的工具软件，我分享下我平时使用到的一些软件。

非神一样的软件

Chrome

当前众多的浏览器有Chrome、IE、Firefox、Opera等等，Chrome吸引我的地方有以下几点：

  • 网页加载速度
  • 强大的插件库
  • 强大的Web调试功能

iTerm

Mac系统自带『Terminal』终端，但是iTerm的功能真是略胜一筹，比如：它的外观、它的标签管理等等

iTerm官网：https://www.iterm2.com/

Atom

这是个强大的开源的编辑器，对，它是开源的，而且它有着丰富的插件库。

Atom官网：https://atom.io/

MacDown

一个开源的，超赞的Markdown文件编辑器，当前这篇文章就是用这个软件写的。

MacDown官网：http://macdown.uranusjr.com/

Sublime

超赞的编辑器，基本是Web开发人员的必备。不过，它是收费的。

Sublime官网：https://www.sublimetext.com/

SourceTree

Git图形化客户端，跟TortoiseGit相比，我更喜欢它。 平时工作时，我会将它与Git命令行结合一起使用

SourceTree官网：https://www.sourcetreeapp.com/

Git

这个代码版本控制工具，是Linus Torvalds大神的另一个杰作。 我想，你没有理由不知道它

Git官网：https://git-scm.com/

Evernote

一个强大的笔记软件

Evernote官网：https://evernote.com/

Alfred

Mac自带Spotlight，这个Alfred除了拥有Spotlight的功能外，它强大的workflow功能真是超赞

Alfred官网：https://www.alfredapp.com/

XMind

一个免费的思维导图软件

XMind官网：http://www.xmind.net/

ToDoist

To Do List 管理软件

ToDoist官网: https://en.todoist.com/

Charles

强大的抓包软件，如果你是客户端开发人员，我想这个软件还是很有用的

Charles官网：https://www.charlesproxy.com/

Dash

一个开发文档搜索工具

Dash官网：https://kapeli.com/dash

Foxmail

一个邮件客户端

Foxmail官网：http://www.foxmail.com/

Keka

一个强大的压缩工具，能够解决Windows压缩文件在Mac上解压出现的乱码问题（字符集不同导致的问题）

KeKa官网：http://www.kekaosx.com/en/

Xcode

Apple公司出品的IDE（支持 C、C++、Objective-C等）

Xcode官网：https://developer.apple.com/xcode/ide/

AppCode

Smart IDE for iOS/OS X development

AppCode官网：https://www.jetbrains.com/objc/

Clion

IDEA公司C、C++开发 IDE

Clion官网：https://www.jetbrains.com/clion/

Android Studio

Android开发IDE

AndroidStudio官网：https://developer.android.com/studio/index.html

Genymotion

一个超赞的Android模拟器

Genymotion官网：http://www.genymotion.cn/

两个神一样的编辑器

有人说Emacs像神的一样的编辑器，也有人说VIM像神一样的编辑器，我说：它们都是有『思想』的编辑器

Emacs

它采用组合键的方式，它的使用直接反应着『函数式编程』的思想。它的『真身』是一个编辑器，但是你可以将它打造成一个IDE。并且，你可以用它来收发邮件、浏览网页等

Emacs官网：https://www.gnu.org/software/emacs/

Vim

它采用三种模式：命令行模式、编辑模式、视图模式，结合这三种模式，你的手可以不用去碰鼠标，全过程用键盘操作。你可以将它打造成一个IDE

Vim官网：http://www.vim.org/
